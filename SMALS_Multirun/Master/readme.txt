This folder contains the SMALS_Multirun workbook and a backup folder. 
Some test data are included. Versions with normalizations for Trend analysis are supplied.
The normalizations differ slightly. 

Routine use: 
None directly. The SMALS_Multirun_Blank sheet should be loaded into the Multirun Bundle folder on the 
desktop, along with PrettyPlot_Multirun, Normasize, and a Readme file. 

Development use: 
Copy the .ASC files to this folder
Develop & test the spreadsheet.
Rename the file in Backup directory
Copy the improved file to Backup directory
