This folder contains:
>Harvest template
>Pretty plot template
 

For daily use in single-run measurements:
Copy the templates into the directory BEFORE you make the run
Save the file
Optionally, perform the 1EXP fits and cut&paste results to cells like G6, G18, G30... 
Then copy & paste the .ASC file into DataDrop

Test data are located in a nearby folder.
For development/testing:
Copy those data into this directory to test and develop the spreadsheet.
Rename existing backup file in backup directory.
Store a new copy of the improved spreadsheet in backup.
